# 仿网易云音乐web

## 项目依赖
- 项目基于React框架编写
- 本项目需要网易云API服务：https://github.com/Binaryify/NeteaseCloudMusicApi. **当前使用版本v2.23.0，有些版本的API接口发生变化会导致项目运行失败**
